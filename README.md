
# Azure Demand Forecasting & Capacity Optimization

## Infosys Internship – Milestone 1 Submission

### Project Objective
Build a data pipeline to collect, clean, and prepare datasets for forecasting Azure Compute and Storage demand to support capacity planning.

---

## Milestone 1: Data Collection & Preparation

### Tasks Completed
• Created Azure compute & storage demand dataset (24 months)  
• Added external economic indicators dataset  
• Cleaned and validated data  
• Merged datasets into a unified dataset  
• Performed feature engineering (month, year, quarter)  
• Generated processed dataset for future modeling
